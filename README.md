# ✈️ Trao AI Travel Planner

An AI-powered travel planning application that generates personalized travel itineraries based on destination, duration and budget.

# Features

- User Registration & Login
- JWT Authentication
- AI Travel Itinerary Generation
- Create and Manage Trips
- View Previous Trips
- Travel Packing Checklist




# Tech Stack

### Frontend
- Next.js
- TypeScript
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication

### AI
- Groq API
- Llama Model
