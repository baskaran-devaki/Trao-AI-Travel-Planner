# ✈️ Trao AI Travel Planner

An AI-powered full-stack travel planning application that helps users create personalized travel itineraries based on destination, duration, and budget.

The application uses AI to generate smart travel plans and allows users to manage their trips with authentication and dashboard features.

---

## 🚀 Features

### 🔐 User Authentication
- User registration
- User login
- JWT based authentication
- Secure user sessions
- Logout functionality

### 🤖 AI Travel Planning
- Generate personalized travel itineraries using Groq AI
- AI suggestions based on:
  - Destination
  - Number of days
  - Budget

### 🗺️ Trip Management
- Create new trips
- Save generated itineraries
- View previous trips
- Personalized user dashboard

### 🎒 Travel Assistant Features
- Interactive packing checklist
- Easy travel preparation management
